1-¿Qué propiedad de CSS se utiliza para convertir el contenedor en un contenedor flex?

La propiedad flex-direction de Flexbox nos permite determinar cuál es el eje principal de nuestro contenedor para, de esta forma, poder jugar con la dirección de los elementos o flex items dentro del contenedor.

2-¿Qué propiedad de CSS se utiliza para centrar los elementos dentro del contenedor a lo largo del eje principal horizontal?

La propiedad de justificación del contenido principal describe cómo sus elementos secundarios se alinean con el elemento principal, utilizando la propiedad justificar-content para alinear todos los elementos secundarios al centro.

3-¿Qué propiedad de CSS se utiliza para centrar los elementos dentro del contenedor a lo largo del eje secundario vertical?

Usamos la propiedad align-items para colocar nuestro elemento en el eje transversal, que en este caso es el eje vertical del bloque, para centrar nuestro cuadro.

4-¿Cuál es la altura del contenedor especificada en el código?

La altura de un elemento CSS se puede especificar usando esta propiedad. La propiedad por defecto establece la altura del área de contenido. La altura del área del borde se decide por el tamaño del cuadro cuando se selecciona el cuadro de borde.
